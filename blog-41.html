⚠️ Important Legal Notice

This guide is for removing password protection from PDFs that you own or have permission to unlock — for example, a PDF you password-protected yourself and want to share more easily, or a document your bank sent you with a known password. Bypassing password protection on documents you don't own or aren't authorized to access may be illegal. This guide only covers scenarios where you already know the password.

PDF passwords are useful for protecting sensitive documents — but they become a nuisance when you need to open the same file repeatedly, share it with colleagues, or merge it with other PDFs. If you own the document and know the password, removing it is straightforward and completely legal.

Two Types of PDF Passwords

PDFs can have two different kinds of password protection, and it's important to know which you're dealing with:

  • Open password (User password) — required just to open and view the document. You must enter this password before you can see any content. This is the most common type.
  • Permissions password (Owner password) — doesn't prevent opening the PDF, but restricts what you can do with it: printing, copying text, editing, etc. You can view the document freely but certain actions are blocked.

The methods below primarily address the Open password (the kind that locks you out of the file entirely). For permissions restrictions, see the dedicated section below.

Remove PDF Password in Chrome (Easiest Method)

This is the quickest and most universal method — it works on Windows, Mac, Linux, and Chromebooks. No extensions or software needed.

  1. Open Google Chrome
  2. Drag and drop the password-protected PDF into a Chrome browser tab, or press Ctrl+O (Cmd+O on Mac) to open it
  3. Chrome will prompt you for the password — enter it
  4. Once the PDF is open in Chrome, press Ctrl+P (Cmd+P on Mac) to open the print dialog
  5. In the printer destination, select "Save as PDF"
  6. Click Save and choose a filename and location
  7. The saved PDF has no password — Chrome printed it to a new unprotected PDF file
✅ This Works Because...

When Chrome prints a PDF to a new file, it renders the document (which you've already authenticated with your password) and saves the rendered output — without the original encryption. It's effectively creating a new PDF from the unlocked content.

Remove Password on Mac — Preview

  1. Open the password-protected PDF in Preview
  2. Enter the password when prompted
  3. Go to File → Export as PDF
  4. In the Save dialog, click Show Details or Security Options
  5. Make sure no passwords are checked
  6. Click Save — the exported PDF has no password

Alternatively, use the Print to PDF method just like Chrome: File → Print → PDF (bottom-left of print dialog) → Save as PDF.

Remove Password on Windows

Method 1: Microsoft Edge (Print to PDF)

Same approach as Chrome: open the PDF in Edge, enter the password, press Ctrl+P, select "Microsoft Print to PDF" as the printer, and save. The output PDF is unprotected.

Method 2: Adobe Acrobat Reader (Free)

  1. Open the PDF in Acrobat Reader and enter the password
  2. Go to File → Print
  3. Select Adobe PDF or Microsoft Print to PDF as the printer
  4. Click Print and save the file — the saved version has no password

⚠️ Don't Upload Sensitive Documents Online

For confidential PDFs — bank statements, legal contracts, medical records — use the Chrome or Mac Preview offline method instead of uploading to an online service. Your file never leaves your device with the Print to PDF approach.

Removing Permissions Restrictions

Some PDFs open without a password but have restrictions on printing, copying text, or editing. These "owner password" restrictions can usually be removed by the Print to PDF method:

  1. Open the restricted PDF in Chrome, Edge, or Preview
  2. Print to PDF (Ctrl+P → Save as PDF)
  3. The saved PDF typically no longer has restrictions — you can now print, copy, and edit it
💡 When This Doesn't Work

Heavily encrypted PDFs with owner passwords (256-bit AES) may not be bypassed by the print method. In that case, use iLovePDF or PDF24's unlock tool, which handle owner password removal more directly. Remember: only remove restrictions from documents you're authorized to modify.

Frequently Asked Questions

Can I unlock a PDF without knowing the password?
This guide only covers removing passwords when you already know them. Bypassing a PDF password you don't know is generally illegal without explicit authorization from the document owner, and is not covered here.
Does the Print to PDF method work on iPhone?
Yes — open the PDF in the Files app or Chrome on iOS, enter the password, then share → Print → pinch to zoom on the print preview → share the resulting PDF. On newer iPhones, you can also use Files app → share → Save to Files to export an unlocked copy.
Will removing the password reduce PDF quality?
The Print to PDF method re-renders the PDF, which can slightly reduce quality for very high-resolution images. For text-only or mostly-text PDFs, quality is identical. For photo-heavy PDFs where quality matters, use iLovePDF or PDF24's direct unlock (which decrypts without re-rendering) for lossless output.
My PDF says "This document is certified" — can I still unlock it?
Certified PDFs have a digital certification that verifies they haven't been modified. Removing the password via Print to PDF creates a new uncertified copy — the certification is removed. If the certification is important (e.g., for legal validity), consult the document's originating organization before unlocking.
