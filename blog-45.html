When to Extract vs Split vs Delete

These three operations sound similar but do different things:

  • Extract pages: Pull specific pages out into a new PDF. Original file stays unchanged. You end up with two files — the original and your extracted subset.
  • Split PDF: Divide a PDF into multiple separate files — usually by page range or splitting every page into its own file. Good for breaking a large document into chapters or sections.
  • Delete pages: Remove pages from a PDF and save the modified version. Good when you want the original file minus a few pages.

Most free tools use "split" to mean both splitting and extracting — they let you specify a page range and download just those pages. The workflow is the same either way.

Method 2 — Browser Print Trick (Zero Tools Needed)

This is the simplest method for extracting a basic page range and works in any browser on any device — no tools, no accounts, no installs.

  1. Open your PDF in Chrome, Firefox, Edge, or Safari (just drag it into the browser window)
  2. Press Ctrl+P (Windows/Linux) or ⌘+P (Mac) to open Print
  3. Change the Destination / Printer to "Save as PDF" or "Microsoft Print to PDF"
  4. In the Pages field, enter your range — e.g. 5-10 or 3, 7, 12-15
  5. Click Save — a new PDF is created containing only those pages

This trick works for any PDF in any browser. The limitation: some PDFs with complex security settings may not print this way. Also, Chrome's print-to-PDF sometimes shifts layout slightly. For pixel-perfect extraction, use a dedicated PDF tool instead.

💡 Works in Any Browser — Widely Unknown

The Print-to-PDF page range trick is one of the least-known free PDF features. It requires absolutely nothing — just a browser you already have. Great for quick one-off extractions when you don't want to open any extra tools.

Method 3 — Mac Preview (Built-In, Drag & Drop)

Mac's built-in Preview app has one of the most intuitive page extraction workflows — you can literally drag pages out with your mouse.

Method A — Drag & Drop to Desktop

  1. Open your PDF in Preview
  2. Show the sidebar: View → Thumbnails
  3. Select the pages you want to extract (click one, then Shift+click for a range, or ⌘+click for individual non-consecutive pages)
  4. Drag the selected thumbnail(s) directly to your Desktop or a Finder window
  5. Preview creates a new PDF containing just those pages

Method B — Delete Unwanted Pages

  1. Open PDF in Preview → show Thumbnails sidebar
  2. Select all pages you don't want (the ones to remove)
  3. Press Delete key
  4. File → Export as PDF — save as a new file (don't overwrite the original)

This approach is faster when you want most pages and only need to remove a few — it's quicker to delete 3 pages than to individually select 47 you want to keep.

Method 5 — iPhone & Android

iPhone — Files App (Built-In)

  1. Open your PDF in the Files app
  2. Tap the Share button (box with arrow)
  3. Tap Print
  4. In the Print Preview, pinch outward on a page thumbnail to expand into a full PDF view
  5. Tap Share again → Save to Files to save the range

Alternatively, use PDF24's website in Safari — it works fully on iPhone without any app installation.

Android — Chrome Browser Trick

The browser Print-to-PDF trick works on Android Chrome as well: open the PDF in Chrome, tap the three-dot menu → Print → select "Save as PDF" → specify the page range. For a dedicated visual tool, Xodo PDF Reader (free) supports page extraction and reorganization with a thumbnail view.

Pro Tips for Page Extraction

  • Always keep the original: Before extracting pages, make sure you have the original PDF saved somewhere safe. Work from a copy if you're also deleting pages from the original.
  • Page numbers vs. PDF page numbers: The page number printed in a document (e.g., "Page 14") may not match the PDF page number (which starts at 1 regardless of document numbering). Check the PDF viewer's page counter, not the printed number in the document header/footer.
  • Non-consecutive pages — use commas: Most tools accept comma-separated values for non-consecutive pages: 1,4,7-10,15 means pages 1, 4, 7 through 10, and 15.
  • Large PDFs: For very large PDFs (500+ pages), browser-based tools like PDFSnap may be faster than online tools because there's no upload time. The processing happens locally regardless of file size.
  • Verify the output: Always open the extracted PDF and check it before sharing or submitting. Count the pages to confirm you got the right range.

Frequently Asked Questions

Will extracting pages reduce the quality of my PDF?
No — extracting pages is a structural operation, not a re-rendering. The extracted pages contain the exact same content, fonts, images, and resolution as the original. Quality is fully preserved. This is different from "converting to PDF" (which re-renders) — extraction just copies the raw page data.
Can I extract pages from a password-protected PDF?
You need to know the open password first. Once you've unlocked the PDF (entered the password to open it), most tools will allow you to extract pages — though some protected PDFs also have permissions restrictions that prevent page extraction. If the tool reports a permissions error, you may need the owner password or to use Adobe Acrobat Pro.
I need to extract every page into separate individual PDFs. Is there a free way?
Yes — PDF24's Split PDF tool has a "Split each page" option that outputs every page as its own PDF file, bundled into a ZIP download. iLovePDF also has this option. PDFSnap's Split tool supports custom ranges which you can use to extract pages one at a time.
How do I extract a page from a PDF on my phone without any app?
Use the browser print trick on Android (Chrome → Print → Save as PDF → specify page range) or open the PDF in Safari on iPhone and use the Share → Print → pinch-to-expand workflow. Both produce a new PDF with just the specified pages and require no apps beyond the browser already on your phone.
