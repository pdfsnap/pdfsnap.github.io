Why Editing PDFs is Tricky

When a Word document is saved as a PDF, the text is no longer stored as editable characters in a simple flow. Instead, it is flattened into a fixed layout — each letter, image, and line is positioned at precise coordinates on the page. Editing a PDF means either modifying these coordinates directly (complex) or overlaying new content on top of the existing content (simpler).

This is why most free PDF editors work by annotation — they add text boxes, highlights, and shapes on top of the original document rather than modifying the underlying content. For most everyday needs — fixing a typo, adding a signature, filling a form, marking up a document — this is completely sufficient. For deep content editing (rewriting paragraphs, changing fonts), you'll need either the original source file or a premium tool.

What You Can Actually Edit for Free

  • Add text boxes: Place new text anywhere on the page — useful for adding notes, filling non-interactive forms, or labelling sections.
  • Highlight and annotate: Highlight text in yellow, underline, strikethrough, and add sticky note comments — standard for document review.
  • Fill interactive forms: PDFs with proper form fields (tax forms, applications, registration forms) can be filled and saved for free using any modern browser.
  • Add a signature: Draw or type your signature and place it anywhere in the document.
  • Add shapes and drawings: Draw arrows, rectangles, and freehand marks for markup and feedback.
  • Delete or reorder pages: Remove unwanted pages or change page order — covered in separate guides on this blog.
ℹ️ What Free Tools Cannot Do Well

Free tools generally cannot reflow existing body text (e.g., change a sentence in the middle of a paragraph and have the text reflow around it). For deep content editing, convert the PDF back to Word first using a free converter, edit in Word, then re-export as PDF.

Mac — Preview App (Best Free Option)

Mac's built-in Preview app is the best free PDF editor available on any platform. It supports text boxes, shapes, signatures, highlights, annotations, and form filling — all without any download or account. Every Mac has it installed.

1

Open the PDF in Preview

Double-click the PDF — it opens in Preview by default. If it opens elsewhere, right-click → Open With → Preview.

2

Open the Markup Toolbar

Click the pencil icon (✎) in the top toolbar, or go to View → Show Markup Toolbar. This reveals all editing tools: text, shapes, sketch, signature, and highlight.

3

Add Text, Highlights, or Signature

Click the T button to add a text box anywhere on the page. Click the highlight tool to highlight selected text. Click the signature button to create and place your signature. All tools are visual and drag-to-position.

4

Save

Press Cmd+S to save. The edits are permanently embedded in the PDF. If you want to keep the original, use File → Export as PDF and save with a new name first.

Windows — Free Options

Microsoft Edge or Google Chrome (Built-in, Free)

Both Edge and Chrome have built-in PDF viewers with basic editing. Open your PDF by dragging it into the browser. In Edge, click the Draw tool (pencil icon) to annotate, add text, or highlight. In Chrome, click the pen icon if visible. Both browsers support filling interactive form fields natively — just click on form fields and type. Save using Ctrl+P → Print to PDF or the download icon.

Adobe Acrobat Reader (Free Download)

The free version of Adobe Acrobat Reader allows highlighting, sticky notes, strikethrough, and filling interactive form fields. It does NOT allow adding text boxes or editing existing text — that requires Acrobat Pro (paid). Download from adobe.com/acrobat/pdf-reader.html. The free tier is sufficient for document review and form filling.

LibreOffice Draw (Free, Full-Featured)

LibreOffice Draw opens PDFs and allows adding text boxes, shapes, and annotations freely. Right-click your PDF → Open With → LibreOffice Draw. Use Insert → Text Box to add text anywhere, or use the shape tools for markup. Export as PDF when done via File → Export as PDF.

iPhone and Android

iPhone — Files App Markup (iOS 15+)

Tap the PDF in the Files app → tap the Share icon → Markup. Or open in Files → tap the pen icon at top right. You get a full markup toolbar: pen, highlighter, text, signature, shapes. All edits save locally. This is a surprisingly capable free editor for mobile.

Android — Adobe Acrobat Mobile (Free Tier)

The free Adobe Acrobat app on Android supports annotation, highlighting, sticky notes, and filling interactive forms. Open your PDF in the app → tap the pen icon → choose your annotation tool. The free tier covers all annotation features — text editing requires a paid subscription.

Free Online PDF Editors

For quick edits from any device without installing anything:

  • Smallpdf Edit PDF — at smallpdf.com/edit-pdf. Add text, images, shapes, and freehand drawings. Free for up to two tasks per day. Clean interface, works well on mobile browsers.
  • iLovePDF Edit — at ilovepdf.com/edit-pdf. Similar feature set. Text boxes, highlights, images, and signatures. Free with no daily limit on the basic annotation features.
  • PDF24 Tools — at tools.pdf24.org/en/edit-pdf. Offers a free online editor plus an optional offline Windows app (PDF24 Creator) for completely local editing without any upload.
⚠️ Online Editors and Sensitive Documents

Online editors upload your file to external servers. For confidential documents — contracts, medical records, legal files — use an offline method: Mac Preview, LibreOffice, Edge/Chrome, or the iOS Files app. These process everything locally on your device.

Frequently Asked Questions

Can I edit text in a PDF for free?
Adding new text boxes on top of existing content is free with Mac Preview, LibreOffice Draw, and online tools. Editing existing body text in a PDF is much harder — it requires rewriting the PDF content stream. The easiest free approach is to convert the PDF to Word (using a free converter), edit the text in Word, and re-export as PDF.
Can I fill a PDF form for free?
Yes — completely free. Any modern browser (Chrome, Edge, Firefox, Safari) can fill interactive PDF form fields without any extension or software. Just open the PDF in the browser, click on form fields, type your answers, and print to PDF to save. No upload, no account, no cost.
How do I add a signature to a PDF for free?
On Mac: open in Preview → Markup Toolbar → Signature tool → draw or photograph your signature → place it on the document. On iPhone: Files app → Markup → Signature. Online: Smallpdf's Sign PDF tool. All are free and create a legally valid electronic signature in most jurisdictions.
Will my edits be permanent in the PDF?
Annotations added by free tools (text boxes, highlights, signatures) are embedded in the PDF and visible to anyone who opens it. However, they are usually stored as an annotation layer, which means some advanced PDF editors can still remove or modify them. For truly locked documents, use a "flatten" option if available, which bakes all annotations into the page permanently.
