Why You Need to Delete Pages from a PDF

  • Removing blank pages from scanned documents: Scanners often insert blank pages between scanned sheets, especially when using automatic document feeders. Deleting these gives a clean, professional result.
  • Sharing only relevant sections: You have a full report but a client or colleague only needs specific sections. Delete everything else before sending to create a focused, smaller document.
  • Removing confidential pages before sharing: A contract PDF has salary information or confidential annexures that should not be shared with certain recipients. Remove those pages before sending the relevant sections.
  • Cleaning up merged PDFs: When merging multiple PDFs, duplicate pages, cover pages, and divider pages often appear in the result. Delete them to create a clean unified document.
  • Reducing file size: Removing unnecessary pages directly reduces PDF file size — often more effectively than compression alone, especially for image-heavy pages.
  • Preparing documents for submission: Many portal submissions have page limits. Delete everything beyond the required page count to meet submission requirements.

Method 4 — Google Chrome Print to PDF (Built-in, No Install)

This is a clever trick that works on any device with Chrome. Open the PDF in Chrome → press Ctrl+P (or Cmd+P on Mac) → change the printer to "Save as PDF" → in the Pages field, type only the pages you want to keep (e.g., "1-4, 8-15") → click Save. Chrome creates a new PDF with only those pages. Free, no upload, works offline.

Method 5 — Mac Preview (Built-in, Free)

Open the PDF in Preview → View → Thumbnails to see all pages in the sidebar → click the page you want to delete → press the Delete key → File → Save. Completely built-in on every Mac, no download needed, no file upload.

Step-by-Step — Chrome Print Method (Works Everywhere)

This method requires no account, no software, and no file upload. It works on any device with Google Chrome installed.

1

Open PDF in Chrome

Drag your PDF file into a Chrome browser tab, or right-click the PDF file and select "Open with Google Chrome".

2

Open Print Dialog

Press Ctrl+P on Windows or Cmd+P on Mac. The print dialog opens showing a preview of your document.

3

Change Destination to Save as PDF

Click "Change" next to the Destination printer. Select "Save as PDF" from the options.

4

Enter Pages to Keep

In the Pages field, enter only the pages you want to keep. For example, if you want to delete pages 3-5 from a 10-page document, type: 1-2, 6-10

5

Save Your New PDF

Click Save and choose where to save the new PDF. It will contain only the pages you specified — effectively deleting the ones you excluded.

⚠️ Important — This Creates a New File

Deleting pages using any of these methods creates a new PDF with the remaining pages. Your original PDF is never modified. Always verify the page count of the new PDF before sharing it to confirm the right pages were removed.

Pro Tips for Page Deletion

  • Always work from a copy. Before deleting pages, make sure you have the original PDF saved somewhere. Accidentally deleting a page you needed means going back to the original — not easy if you have overwritten it.
  • Use thumbnail view before deleting. Tools like iLovePDF show page thumbnails, which helps you visually confirm you are deleting the correct pages rather than relying on page numbers alone.
  • For removing blank pages in bulk, consider using PDF24's organizer which lets you visually identify and delete all blank pages at once rather than one by one.
  • Compress after deleting. After removing pages, run the resulting PDF through PDFSnap's Compress PDF tool. Page deletion reduces page count but the remaining pages' image data remains — compression further reduces the final file size.

Frequently Asked Questions

Can I delete pages from a PDF without any software?
Yes — the Chrome browser trick works without any additional software. Open your PDF in Chrome, press Ctrl+P, select Save as PDF, type only the page numbers you want to keep, and save. This works on any device with Google Chrome installed — Windows, Mac, Linux, and Android.
Will deleting pages affect the quality of remaining content?
No — deleting pages removes those pages completely from the PDF but does not affect the quality of the remaining pages in any way. Text, images, and formatting on the remaining pages are completely unchanged.
Can I delete pages from a scanned PDF?
Yes — scanned PDFs are just image-based PDFs. Page deletion works the same way regardless of whether the PDF contains text or scanned images. The blank pages common in scanned documents can be deleted just like any other page.
Is there a limit to how many pages I can delete?
No practical limit exists for any of the methods described. You can delete a single page or hundreds of pages from a document. The only constraint is that at least one page must remain in the PDF — you cannot create an empty PDF by deleting all pages.
Can I undo page deletion after saving?
No — once you save the new PDF with pages removed, there is no undo. The only way to restore deleted pages is to go back to your original PDF file. Always keep the original before making any modifications to a PDF.
