# PDFSnap — Free PDF & Image Tools

**PDFSnap** is a free, browser-based PDF and image tool suite available at [https://pdfsnap.github.io](https://pdfsnap.github.io)

## What it does

- 🗂️ Merge, split, rotate, compress PDFs
- 🖼️ Convert PDF to JPG/PNG and images to PDF
- 🔒 Password protect, unlock, sign PDFs
- 📐 Compress, resize, crop, convert images
- 🤖 AI PDF summarizer (100% free)
- 📱 QR Code generator

## How it works

All processing runs **entirely in your browser** using [pdf-lib](https://github.com/Hopding/pdf-lib) and [PDF.js](https://github.com/mozilla/pdf.js). Your files never leave your device. No server, no signup, no watermarks.

## Built with

- pdf-lib 1.17.1
- PDF.js 3.11
- Tesseract.js (for OCR)
- Vanilla JavaScript — no frameworks
